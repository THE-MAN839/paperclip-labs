## Key Concepts from the Skill

### 1. Write-Ahead Logging (WAL)

**Problem:** Context loss at compaction

**Solution:** Write to `SESSION-STATE.md` BEFORE context fills

**Flow:**
```
Active work → SESSION-STATE.md (WAL)
          ↓
Compaction (context resets)
          ↓
Read SESSION-STATE.md → Continue work
```

**This means I won't lose context when compaction happens.**

---

### 2. Danger Zone Protocol

**When context >60%:**
- Activate working buffer
- Log EVERY exchange (human summary + agent summary)
- After compaction: Read buffer FIRST → Extract → Continue

**This ensures continuity across sessions.**
